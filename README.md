# AI Resume Screening & Interview Intelligence System

An AI-powered Resume Screening System built using Retrieval-Augmented Generation (RAG), FAISS, Sentence Transformers, Streamlit, and Hugging Face Transformers.

## Features

- Resume PDF Upload
- Resume Text Extraction
- Semantic Search using FAISS
- AI-Powered Resume Question Answering
- Resume Chunking and Embeddings
- Candidate Skill Analysis
- Intelligent Resume Retrieval
- Streamlit Web Interface

---

## Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- Sentence Transformers
- Hugging Face Transformers
- PDFPlumber

---

## Project Architecture

```text
Resume PDF
    ↓
Text Extraction
    ↓
Chunking
    ↓
Embeddings Generation
    ↓
FAISS Vector Database
    ↓
Semantic Search
    ↓
AI Response Generation
```

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Resume-Screening-System.git
```

### Navigate to Project

```bash
cd AI-Resume-Screening-System
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
streamlit run app.py
```
